# Ip_oracle
IP Oracle is a fast and simple Python-based tool that validates an IP address and retrieves detailed information such as location, network, and organization using an external API. It features a clean terminal interface with colored output and a stylish banner✨

# Feature 
<ul>
<li>IP address validation</li>
<li>Fetch detailed IP information</li>
<li>Clean and colorful terminal output</li>
<li>Stylish ASCII banner</li>
<li>Lightweight and beginner-friendly</li>
</ul>

<hr>

# Overview 
<table>
<tr><th>Field</th><th>Description</th></tr>
<tr><td>IP</td><td>Target IP address</td></tr>
<tr><td>VERSION</td><td>IPv4 or IPv6</td></tr>
<tr><td>COUNTRY</td><td>Country of the IP</td></tr>
<tr><td>REGION</td><td>Region or state</td></tr>
<tr><td>TIMEZONE</td><td>Timezone</td></tr>
<tr><td>ORG</td><td>Organization (ISP/company)</td></tr>
<tr><td>ASN</td><td>Autonomous System Number</td></tr>
</table>
